# SchoolPrepar - Plateforme d'Orientation Scolaire

## 1. Présentation du Projet
SchoolPrepar est une solution numérique (Web et Mobile) conçue pour centraliser les informations sur l'orientation scolaire et professionnelle. Elle permet aux élèves de découvrir des filières, des établissements, et d'échanger avec des mentors pour faire des choix éclairés.

## 2. Objectifs Pédagogiques (TP 1)
Ce projet est réalisé dans le cadre de l'UE IT 232 - Développement Web II.
* Installation et configuration du framework Symfony.
* Compréhension et mise en place de l'architecture MVC.
* Analyse fonctionnelle et modélisation (Cas d'utilisation et ERD).

## 3. Architecture Technique
* **Framework :** Symfony 6/7
* **Moteur de template :** Twig (Layout principal structuré)
* **Contrôleurs :** - `MainController` : Gestion de la page d'accueil dynamique.
    - `SchoolController` : Gestion des fiches établissements.
* **Gestionnaire de versions :** Git

## 4. Installation et Lancement
Pour exécuter ce projet localement :

1.  **Clonage du dépôt :**
    ```bash
    git clone [https://github.com/Edmund1er/symfony_schoolprepar_project_roms.git](https://github.com/Edmund1er/symfony_schoolprepar_project_roms.git)
    ```
2.  **Installation des dépendances :**
    ```bash
    composer install
    ```
3.  **Démarrage du serveur :**
    ```bash
    symfony server:start
    ```
